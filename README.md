# Aadhaar System Health Monitoring Dashboard

## Overview
This project analyzes Aadhaar enrolment and update activity at the district level
to identify system load and coverage gaps.

## Methodology
- Load Score = Total Updates / Total Enrolments
- Coverage Gap based on relative enrolment shortfall
- Districts classified as Healthy or Under-covered

## Tools Used
- Python (pandas, numpy)
- Jupyter Notebook
- Tableau

## Reproducibility
All data processing is implemented in Python notebooks.
Tableau dashboards are built using the processed outputs.
